# Test Plan: US-04 (View My Onboarding Journey)

**Date:** 2026-03-18
**Story:** US-04: View My Onboarding Journey

## 1. Test Objectives
Validate that the "My Journey" view accurately renders onboarding milestones grouped into specific sections, displays the correct milestone details (timeframe, estimated time), calculates overall progress accurately, and correctly identifies overdue tasks based on the user's start date.

## 2. Test Cases

### TC-01: Navigation & Layout
- **Preconditions:** User is logged in, has selected a role, and completed the skills assessment.
- **Steps:** Navigate to "My Journey" via the sidebar.
- **Expected Result:** The Journey map is displayed. There are three collapsible sections:
  1. Week 1: Foundations
  2. Week 1-2: Core Systems
  3. Week 1-2: Role-Specific Training
- **Status:** Pending

### TC-02: Overall Progress Calculation
- **Preconditions:** Journey view is open.
- **Steps:** Check the progress bar at the top summary card. Toggle a milestone to complete.
- **Expected Result:** The circular progress ring and percentage should update proportionally out of the total required for that user's role.
- **Status:** Pending

### TC-03: Milestone Card Information
- **Preconditions:** Journey view is open.
- **Steps:** View a specific milestone card (e.g., "Set up Everest email"). Expand the card.
- **Expected Result:** 
  - Standard view shows: Title, Category Badge, Calendar icon with Timeframe (e.g. "By Day 1"), Clock icon with Estimated Time (e.g. "10 mins").
  - Expanded view shows: Brief Description/Instruction snippet.
- **Status:** Pending

### TC-04: Overdue Highlighting Logic
- **Preconditions:** User `joinedAt` date is set to more than 7 days ago.
- **Steps:** View Phase 1 (Week 1) milestones that are unchecked.
- **Expected Result:** Unchecked Week 1 milestones display a red border and a small red "Overdue" badge with an `AlertCircle` icon.
- **Status:** Pending

### TC-05: Non-Overdue Logic
- **Preconditions:** User `joinedAt` date is set to "today".
- **Steps:** View Phase 1 and Phase 2 milestones.
- **Expected Result:** No red borders or Overdue tags appear. Standard styling applies.
- **Status:** Pending
