# Onboarding Companion

A smart, AI-driven onboarding assistant designed to reduce time-to-productivity for new hires.

## Features
- **Role-Based Dashboard:** Personalized view based on user role (e.g., Backend Engineer).
- **Journey Map:** Visual timeline of the onboarding progress.
- **Context-Aware Resources:** Curated documentation links, filtering out obsolete pages.
- **Team Directory:** Quick access to mentors and key contacts.

## Tech Stack
- **Framework:** React + Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide React

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Locally**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
